# [기본 가격표] (display 기준 절반 아래)
## ▣ input
### ☆ 기본 (행사 유무와 관계 없음)
#### 1. 제품명 - 입력 "---"
#### 2. g당 가격 표기 - 입력 "---"g당 "---"원
#### 3. 가격 - 입력 "---"
### ★ 행사 
##### 4. 버튼형 세일 이미지 옵션 선택 - [특가], [마감], [1+1], [2+1]로 버튼 형식
###### ① 특가 sale : 가격 세일의 경우 기본 이미지
###### ② 마감 sale : 행사일 중 마지막 날일 때 사용
###### ③ 1+1 sale : 1+1
###### ④ 2+1 sale : 2+1
##### 5. 행사 전 가격 - 입력 "---"
##### 6. 행사 날짜 - 입력 "---" . "---" . "---" ~ "---" . "---" . "---"
